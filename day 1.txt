변수 - 하나의 값을 저장할 수 있는 메모리 공간

변수 타입 - 기본형, 참조형(ex 클래스)

변수 명명규칙

    - 대소문자 구분
    - 예약어 불가
    - 숫자로 시작 불가
    - 특수문자는 '_', '$'만 허용($은 첫글자만 허용)

프로그래머 관례

    - 클래스 이름의 첫 글자는 항상 대문자
    - 여러 단어로 이루어진 이름은 단어의 첫 글자를 대문자로
    - 상수의이름은 모두 대문자로 하고 여러 단어로 이루어진 경우 '_'로 구분

자료형

    - 논리형 boolean
    - 문자형 char(UTF 16이라서 2byte)
    - 정수형 byte, short, int, long
    - 실수형 float(float의 경우 대입값의 뒤에 f쓰기), double

상수 && 리터럴

    - 상수는 고정된 값으로 앞에 변수 앞에 final을 붙인다.
    - 리터럴은 값 자체로 의미를 가지는 것을 의미
        ex) int day =  x;
                     리터럴

자료형변환

    - 변수 || 상수의 타입을 다른 타입으로 변환하는 것
    - 자료형 변환법 -> (변환할 타입)피연산자
        ex) double x = 3.14;
             int y = (int)x;
    - 자동형 변환의 경우 표현범위가 좁은 쪽에서 큰 쪽으로 변환할 경우 해당
